
  Quick Usage Guide

  For JavaScript projects:
  bun install          # Install dependencies
  bun add package      # Add new package
  bun run dev          # Run scripts

  For Python projects (recommended workflow):
  # Create virtual environment
  uv venv

  # Activate it
  source .venv/bin/activate

  # Install packages super fast
  uv pip install requests pandas numpy

  # Or create a new project with uv
  uv init my-python-project

  Switch Python versions:
  pyenv install 3.12.0    # Install a version
  pyenv global 3.12.0     # Set globally
  pyenv local 3.11.0      # Set for current directory
